# MyFrame is a lightweight message-based and event-driven application framework
	
## Build

```sh
	mkdir build
	cd build
	cmake ../
	make
```

## Run

```sh
	cd bin
	./myframe
```


## 概述
	- 使用线程池实现的应用程序框架，支持高并发
	- 基于事件驱动的程序框架
	- 支持线程间无锁通信，提高并发性能
    - 基于服务的编程模式
    - 仅支持linux
	- No document (you can find more docs in the code)
